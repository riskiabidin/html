#HTML Layout Elements

  • header:defines a header for a document or a section
  • nav:defines container for navigation links
  • section: defines a section in a document
  • aside:defines content aside form the content (like a sidebar)
  • footer:defines a footer for a document or a section
  • details:defines additional details
  • summary:defines a heading for detail elemen

#HTML Layout Techniques

There are four different techiniques to create multicolumn layout. Each way has its pros and cons:

  • CSS framework
  • CSS float property
  • CSS flexbox
  • CSS grid
