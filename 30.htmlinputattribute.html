HTML Input Attributes

This chapter describes the different attributes for the HTML input element.

#The Value Attribute

The input value attribute specifies an initial value for an input field:




#The readonly attribute

The input readonly attribute specifies that an input field is read only.

A rad-only input field cannot be modified (however, a user can tab to it, highlight it and copy the text for it).

The value of a read-only input field will be sent when submitting the form!





#The disabled attribute

The input disable attribute specifies that an input field should be disabled

A disabled input field is unusable and un-clickable

The value of a disabled input field will not be sent when submitting the form!




#The size attribute

The input size attribute specifies the visible width, in characters of an input field. The defaul value for size is 20.

Note: The size attribute works with the following input types:text, search, tel, url, email and password.




#The maxlength attribute

The input maxlength attribute specifies the maximum number of characters allowed in an input field.

Note: When a maxlength is set, the input field will not wccpet morer than the specified number of characters. However, this attribute dows not provide any feedback. So, if you wan to alert the user, you must write javascript code.




#The min and max attribute

The input min and max attribute specify the minimum and maximum values for an input field.

The min and max attribute work with the followng input types:number, range, date, datetime-local, month, time and week.

Tip:use the max and min attributes together to create a range of legal values.



#The multiple attribute

The input multiple attribute spsecifies that the user is allowed to enter more than one value in an input field.

The multiple attribute works with the followng input types:email and file.

#The pattern Attribute

The input pattern attribute specifies a regular expressin that the input field's value is checked against, when the form is submited

The pattern attribute works with the following input types:text,date, search, url, tel, email and password.

Tip: Use the global title attribute to describe the pattern to help the user.

Tip: Learn more about regular expressions in our javascript tutorial.



#The placeholder Attribute

The input placeholder attribute specifies short a hint that describes the expected value of an input-field (a sample value or a short description of the espected format).

The short hint is displayed in the input field before the user enters a value.

The placeholder attribute works with the following input types:text, search, url, tel, email and passord.

#The required attribute

The input required attribute specifies that an input field must be filled out before submitting the form.

The required attribute works with the following input types:text, serarch, url, tel, email, password, date pickers, numberm checkbox, radio and file.



#The step attribute

The input step attribute specifies the legal number intervals for an input field.

Ecample:if step="3", legal numbers could be -3, 0, 3, 6, etc.

Tip: This attribute can be used together with the max and min attributes to create a range of legal values.

The step attribute works with the following input types:number, range, date, datetime-local, month, time and week.

Note: Input restrictions are note foolproof and javacript provides many ways to add ilegal input. To safely restrict input, it must also be checked by the receiver (the server)!

#The autofocus attribute

The input autofocus attribute specifies that an input field should automatically get focus when the page loads.





#The height and width attribute

The input heght and width attributes specify the height and width of an input type="image" element

Tip: Always specify both the height and width attribute for images. if height and width are set, the space reuired for the image is reserved when the page is loaded. Without these attribute, the browser does not know the size of the image and cannot reserve the appropriate space to it. the effect will be that the page layout will change during loading (while images load).





#The list attribute

The input list attribute refers to a datalist eleent that contains pre-defined option for an input element.

#The autocomplete attribute

The input autocomplete attribute specifies whether a for or an input field should have autocomplete on or off.

Autocomplete allows the browser to predict the value. Whwn a user start to type in a field, the browser should display option to fill in the field, based on earlier typed value.

The autocomplete attribute works with forn and the following input type:text, search, url, tel, email, password, datepicker, range and color.
