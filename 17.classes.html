HTML The class Attribute

An HTML element can belong to one or more classes. Classes are used by CSS and jacaScript to select and access spesific elements.

#Using The class Attribute

The HTML class attribute specifies one or more class names for an element.

The class attribute is often used to point to a class in a style sheet. it can also be used by a javascript to access and manipulate elements with the specific class name(s)

In the following example we have theree div elements that point to the same class name ("city"). All of three div elements will be styled equally according to the .city style definition in the head section:

#Multiple classes

HTML elements can have more than one class name.

To specify multiple classes, separate the class names with a space, e.g div class="city main". This allows you to combine several CSS classes for one HTML element
